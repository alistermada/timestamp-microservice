API Basejump: Timestamp microservice

Checks to see if the string passed as a parameter is a valid unix timestamp or a natural language date.

If it's valid, return both the unix timestamp and natural language date.

If not, returns {"unix":null,"natural":null}

Example usage:

https://timestamp-micro-api.herokuapp.com/January 1, 2016
https://timestamp-micro-api.herokuapp.com/1451581200

Example output:

{"unix":1451581200,"natural":"January 1, 2016"}
